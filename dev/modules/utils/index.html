Utilities

IgDiscover.from_toml — Method
from_toml(T, d::Dict) -> T

Construct a struct T from a TOML dictionary by matching field names to keys. Each field is converted to its declared type. Nested structs are constructed recursively via dispatch when the value is a Dict.
