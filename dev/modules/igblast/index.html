IgBLAST

IgBLAST assigns V, D, and J gene segments to each read. This module wraps the external igblastn and makeblastdb programs: it builds BLAST databases from the germline FASTA files and runs IgBLAST on read chunks (optionally in parallel).

IgDiscover.make_blastdb — Method
make_blastdb(fasta_path, db_name; prefix="%")

Create a BLAST database. The % prefix avoids IgBLAST GenBank-style name mangling.

source
IgDiscover.make_vdj_blastdb — Method
make_vdj_blastdb(blastdb_dir, database_dir)

Build V, D, J BLAST databases in blastdb_dir from FASTA files in database_dir.

run_igblast_chunk(sequences, blastdb_dir, penalty; species, sequence_type, threads) -> String

Run igblastn on sequences with a mismatch penalty.

IgDiscover.run_igblast_chunk — Method
run_igblast_chunk(sequences, blastdb_dir; species, sequence_type, threads) -> String

Run igblastn on sequences (no mismatch penalty override). Uses IgBLAST's internal threading (-num_threads) for parallelism.

source
IgDiscover.run_igblast_on_fasta — Method
run_igblast_on_fasta(database_dir, input_fasta, output_tsv_gz;
                    sequence_type, species, penalty, threads, limit, chunksize)

Run IgBLAST on a FASTA file, write AIRR TSV (gzipped) output. Parallelism is via IgBLAST's internal threading (-num_threads). Chunks are run sequentially (one process at a time); chunking only limits memory per process.
