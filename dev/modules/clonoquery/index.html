Clonoquery

IgDiscover.CDR3CoreSlice — Type
CDR3CoreSlice(start, stop)

Define the non-junction region of CDR3 sequences. Positions before start and after stop are junction regions; at least one junction must be identical for two CDR3s to be considered similar.

source
IgDiscover.ClonoQuery — Type
ClonoQuery(; mismatches=1.0, cdr3_core=nothing, use_aa=false, min_count=1)

Callable struct that queries a reference table by clonotype.

IgDiscover.is_similar_with_junction — Method
is_similar_with_junction(s, t, mismatches, core::CDR3CoreSlice) -> Bool

CDR3 similarity with junction constraint: Hamming within threshold and at least one junction region (before core.start or after core.stop) is identical between s and t.

IgDiscover.is_similar_with_junction — Method
is_similar_with_junction(s, t, mismatches, core::Nothing) -> Bool

CDR3 similarity without junction constraint: same length and Hamming ≤ threshold. When mismatches ∈ (0,1), it is a fraction of CDR3 length.
