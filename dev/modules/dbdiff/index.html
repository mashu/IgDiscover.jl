Database Comparison

IgDiscover.DatabaseComparator — Type
DatabaseComparator(max_cost=20)

Callable struct comparing two FASTA record sets. Pairs identical sequences first, then uses greedy minimum-cost matching for remaining records.

cmp = DatabaseComparator()
diff = cmp(read_fasta("a.fasta"), read_fasta("b.fasta"))
